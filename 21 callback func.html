콜백 함수

함수(method)의 입력값(인자)로 다른 함수가 사용될 떄, 인자 함수를 콜백 함수라고 함 콜백 함수는 간단한 표현, 즉 화살표 함수 형식을 많이 사용함.

  • 요소.addEventlistner('이벤트 이름', 콜백함수)
  • 배열.forEach(콜백함수) - 배열 요소를 하나씩 가져다가 콜백함수 실행
  • 배열.sort(콜백함수) - 배열요소를 2개씩 가져다가 값을 비교하는 콜백함수
  • window.setTimeout(콜백함수,ms시간) - 정해진 시간 지나고 1번만 콜백 함수 실행
  • window.setinterval(콜백함수,ms시간) - 정해진 시간 간격을 가지고 함수를 반복 실행
개발자가 정의하는 함수의 인자를 콜백함수로 만들 수도 있음
